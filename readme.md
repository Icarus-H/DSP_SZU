# DSP_SZU
深圳大学DSP课程实验代码 个人使用兼存档

## 来源
[深圳大学 机电学院 2022年春 DSP课程实验 DSP_Labs of CMCE.](https://github.com/team0ne1/SZU_DSP_Labs)

## 环境
Code Composer Studio Version: 11.2.0.00007

OS: Windows 10

## 程序与实验对照
demo1:(3) LED和跑马灯 按键停止与继续跑马灯

demo2:(5) 矩阵键盘

demo3:(1) 串口与SPI实验 数码管

demo4:(2) 中断管理和CPU计数

demo5:(3) LED和跑马灯 矩阵键盘选择功能

demo6:(4) 看门狗

demo7:(6) 作弊秒表

demo8:(7) 模拟信号AD采集, 0-3V电压表; 使用扫描法

demo9:(7) 模拟信号AD采集, 0-3V电压表; 使用中断

## 实验列表
* 串口与SPI实验
* 中断管理和CPU计数
* LED和跑马灯
* 看门狗
* 矩阵键盘
